# 📚 三科在线答题系统

> 学校期末考试复习专用，所有题目均来自试卷真题，考试形式为线上，题目会从本系统中抽取。

## 📋 涵盖科目

- 🌐 **计算机网络**（填空题、判断题、选择题、简答题、综合题）
- 🌳 **数据结构**（填空题、判断题、选择题、问答题、算法设计题）
- ⚙️ **操作系统**（选择题、判断题、问答题、综合题）

## 🎯 功能特点

- ✅ **选择题**：点击选项自动判断对错，多选可提交答案
- ✅ **判断题**：点击“正确/错误”按钮即时反馈
- ✅ **填空题**：输入文字后提交，忽略空格大小写比较
- ✅ **简答题/综合题**：输入文字后对照标准答案
- ✅ **显示/隐藏答案**：每道题都可以查看标准答案和解析
- ✅ **科目筛选**：按单一科目复习
- ✅ **关键词搜索**：快速定位题目
- ✅ **随机抽题**：碎片时间随时刷
- ✅ **夜间模式**：护眼背诵
- ✅ 部署于 **GitHub Pages**，手机/电脑/平板均可访问
- ✅ 现增加了筛选功能，根据每科题目的题型方便于背诵
